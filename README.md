## 👨‍💻 Sobre Mim

Estudante de Gestão de Tecnologia da Informação pela FATEC, sou apaixonado por análise de dados e Business Intelligence. Busco constantemente aprimorar minhas habilidades técnicas para transformar dados em insights valiosos para tomada de decisões. Meu foco está em desenvolver soluções práticas que conectem tecnologia e negócios, aplicando conhecimentos em SQL, visualização de dados e processos ETL para resolver problemas reais.
